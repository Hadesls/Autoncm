# 具有GUI界面的ncm转flac\mp3格式工具：Autoncm

使用本程序可将下载的网易云音乐VIP下载的文件（ncm）转换为 mp3 或 flac 格式，纯Python编写。也可以自行转换为exe。
ui界面采用flet的UI库。
# 使用 Python格式
需要配置Python环境，



# 使用 exe
无需配置Python环境，仅支持Windows 10 以上操作系统。

直接下载**Autoncm.exe**，打开后点击**开始转换**直接使用。
